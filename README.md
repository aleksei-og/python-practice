# Python Practise

Этот репозиторий содержит серию практических работ по Python.  
Каждая работа оформлена в формате Jupyter Notebook и направлена на постепенное развитие навыков программирования, работы с данными и решения прикладных задач.

## Состав проекта

| Файл | Описание |
|-----|----------|
| **Oganov_DZ1_MIVT_24_3_1.ipynb** | Базовые конструкции Python: переменные, ввод/вывод, условия, циклы. |
| **Oganov_DZ2_MIVT_24_3_1.ipynb** | Функции, параметры, обработка строк и простые алгоритмические задачи. |
| **Oganov_DZ3_MIVT_24_3_1.ipynb** | Работа со списками и словарями, перебор структур данных, задачи на обработку коллекций. |
| **Oganov_DZ4_MIVT_24_3_1.ipynb** | Работа с файлами: чтение, запись, простейшая обработка данных в текстовых файлах. |
| **Oganov_final_MIVT_24_3_1.ipynb** | Итоговая комбинированная работа: применение функций, структур данных и обработки данных для решения комплексной задачи. |

## Требуемые библиотеки

Работы используют только стандартную библиотеку Python.  
Дополнительные пакеты **не требуются**.
