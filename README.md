# Python Practise

Данный репозиторий содержит практические работы по Python, выполненные в рамках обучения.  
Цель проекта — отработка базовых и прикладных навыков программирования: работа со строками, структурами данных, файлами, массивами, визуализацией и алгоритмами сортировки.

## Структура проекта

| Файл / Папка | Описание |
|-------------|----------|
| `Oganov_DZ1_MIVT_24_3_1.ipynb` | Основы Python: переменные, типы данных, ветвления, циклы. |
| `Strings_Oganov_Alexey_MIVT_24_3_1.ipynb` | Работа со строками: поиск, индексация, частотный анализ, обработка backspace (`@`). |
| `NumPy_Oganov_Alexey_MIVT_24_3_1_.ipynb` | Работа с массивами с помощью **NumPy**: создание, индексация, преобразование, статистика. |
| `Matplotlib_Oganov_Alexey_MIVT_24_3_1.ipynb` | Визуализация данных: построение графиков по отсортированным данным. |
| `Sorting_Oganov_Alexey_MIVT_24_3_1.ipynb` | Реализация алгоритмов сортировки + сравнение по времени (bubble / selection / insertion). |
| `Oganov_DZ2_MIVT_24_3_1.ipynb` | Дополнительные задачи по циклам, логике и функциям. |
| `Oganov_DZ3_MIVT_24_3_1.ipynb` | Работа со списками и словарями, обработка данных. |
| `Oganov_DZ4_MIVT_24_3_1.ipynb` | Работа с файлами, обработка данных, чтение/запись. |
| `Oganov_final_MIVT_24_3_1.ipynb` | Итоговая работа: комбинирование навыков, решение комплексной задачи. |
| `README.md` | Описание проекта (вы сейчас здесь). |
